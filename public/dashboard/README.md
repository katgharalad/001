# US GHG Emissions Control Panel

Dark mode, neon-accented analytics dashboard for EPA GHGRP emissions data (2010-2023).

## Features

- **Hero Chart**: Interactive time trend visualization with year-over-year comparisons
- **KPI Cards**: Four stacked metric cards with gradient backgrounds
- **State Leaders**: Mini bar chart showing top emitting states
- **Progress Tracking**: Decarbonization progress sparkline
- **Interactive Filters**: Timeline-based filter system in right panel
- **Year Navigation**: Navigate through 2010-2023 data
- **Responsive Design**: Adapts to different screen sizes

## Running the Dashboard

Due to browser security restrictions, CSV files cannot be loaded directly from `file://` protocol. You need to run a local web server.

### Option 1: Python HTTP Server

```bash
cd dashboard
python3 -m http.server 8000
```

Then open: `http://localhost:8000`

### Option 2: Node.js HTTP Server

```bash
cd dashboard
npx http-server -p 8000
```

Then open: `http://localhost:8000`

### Option 3: VS Code Live Server

If using VS Code, install the "Live Server" extension and right-click `index.html` → "Open with Live Server"

## Data Requirements

The dashboard expects CSV files in `../data_processed/`:
- `ghg_all_years_clean.csv`
- `ghg_state_year.csv`
- `ghg_sector_year.csv`
- `ghg_facility_clean.csv`

These are generated by running `run_pipeline.py` from the project root.

## Customization

### Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --accent-primary: #C6FF3F;
    --accent-teal: #4DE2C2;
    --bg-body: #101116;
    /* ... */
}
```

### Filters

Add new timeline filters in `index.html`:

```html
<div class="timeline-item" data-filter="your-filter">
    <!-- Filter content -->
</div>
```

Then implement filter logic in `dashboard.js` `updateDashboard()` method.
